This document describes conditions under which states,
relations, or configurations return or reappear.

It does not assume correction, resolution, or improvement.

---

## Triggers
- saturation
- loss of relation
- delay
- external interruption

## Forms of Return
- repetition
- drifted recurrence
- partial reappearance
- inversion

## Notes
- return does not imply restoration
- original state may not be preserved
- absence can also function as return
